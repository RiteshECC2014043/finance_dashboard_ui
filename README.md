## ✨ Key Features
- **Role-Based Access Control (RBAC):** - **Admin View:** Access to security logs, user management, and global transaction data.
  - **User View:** Personal expense tracking and profile settings.
- **Dynamic Dark Mode:** Toggle between light and dark themes for a personalized experience.
- **Transaction Management:** Filter, search, and categorize financial records.
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile screens using CSS Variables.

## 🛠️ Tech Stack
- **Frontend:** React
- **Styling:** CSS3 (Custom Properties / Flexbox / Grid)
- **Deployment:** Netlify
- **Version Control:** Git & GitHub
